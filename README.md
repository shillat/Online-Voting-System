# Online-Voting-System
Full-stack online voting system designed to replace traditional paper ballots. The application features role-based access control (RBAC), separating Admin management tools (voter registration, election setup) from the secure Voter dashboard. Built with a modern React frontend (Tailwind CSS) and RESTful API integration.
